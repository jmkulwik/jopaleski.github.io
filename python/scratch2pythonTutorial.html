Part 2: Command Line!

Open your command prompt

  1. Click on the Start menu
  2. Type cmd. Click on cmd. this will open the command prompt.

Open Window's Explorer (not Internet Explorer)

  1. Click on the Start menu
  2. On the right panel, click "Computer"
  3. Find where you are on in the command prompt - (you probably need to click on the C drive, then Users, then your username).

Explore your files and folders

  1. In the command prompt, type dir - this will show you which files and folders are in your current directory
  2. Make a new directory called "Documents". Type mkdir Documents
  3. Type dir again. Notice that Documents is now listed (if you click the refresh button in Window's Explorer, you should see Documents appear)
  4. Type cd Documents. You are now inside the Documents folder

Dive Into Python

  1. Inside the command prompt type [Ctrl-Z] [Enter] followed by notepad++ firstProgram.py. This should open a new window.
  2. Type some of the same commands you ran for the Intro section. Each command should be on a new line.
  3. Pick a command you want to see the result of and put it in the print function. i.e. if you want to see the answer to 3+4, type print(3+4)
  4. To run your code, go to the command prompt and type python firstProgram.py. Make sure you save first!!
